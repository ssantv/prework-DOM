Document Object Model

Clase 4 — JavaScript

DOM son las siglas de Document Object Model. Cuando el navegador carga el HTML, no lo lee como texto plano. Lo convierte en un árbol de objetos en memoria. Ese árbol es el DOM.

document
  └── html
      ├── head
      │   └── title
      └── body
          ├── h1
          ├── p
          └── button

Cada etiqueta se convierte en un objeto de JavaScript. Con propiedades. Con métodos. Que podéis leer y modificar.

// Este h1 en HTML:
<h1 id="titulo">Hola mundo</h1>

// JS lo ve como:
{
  id: "titulo",
  textContent: "Hola mundo",
  style: { ... },
  classList: [ ... ]
}
💡 Clave: Si podéis modificar un objeto, podéis modificar la página.

Antes de tocar nada hay que seleccionar el elemento. Igual que en CSS primero apuntáis al elemento y luego le aplicáis algo.

getElementById / getElementsByClassName / getElementsByTagName
const titulo = document.getElementById("mi-titulo");
const parrafos = document.getElementsByTagName("p");
const cajas    = document.getElementsByClassName("caja");
querySelector — el más flexible
document.querySelector("h1");          // por etiqueta
document.querySelector("#mi-titulo"); // por id  (con #)
document.querySelector(".intro");     // por clase (con .)

// Devuelve solo el PRIMERO que encuentre
querySelectorAll — todos los que coincidan
const todos = document.querySelectorAll("p");
// → NodeList, como un array
Error clásico: Si el elemento no existe, devuelve null. Intentar modificar null rompe el programa. 
const noExiste = document.querySelector(".no-existe");
noExiste.textContent = "Hola"; // TypeError

Una vez seleccionado el elemento, podemos leer o cambiar lo que muestra.

textContent
const p = document.querySelector("p");

p.textContent = "Texto nuevo";   // cambiar
console.log(p.textContent);      // leer

// ✅ Trata todo como texto plano
// Si metes HTML, lo muestra literalmente
innerHTML
const div = document.querySelector("div");

// Interpreta las etiquetas HTML
div.innerHTML = "<strong>Negrita</strong> y <em>cursiva</em>";

// Usa textContent para texto normal
// Usa innerHTML solo si necesitas meter etiquetas

Dos formas de cambiar cómo se ve un elemento: estilos directos con .style, o clases CSS con classList.

.style — estilos inline
const caja = document.querySelector(".caja");

caja.style.color           = "red";
caja.style.backgroundColor = "#1a1a1a"; // ← camelCase
caja.style.fontSize        = "2rem";
caja.style.display         = "none";    // ocultar

// ⚠️ En CSS escribís background-color (con guion)
// En JS no podéis usar guiones → backgroundColor
classList — la forma recomendada
const btn = document.querySelector("button");

btn.classList.add("activo");         // añadir
btn.classList.remove("activo");      // quitar
btn.classList.toggle("activo");      // si está la quita, si no la pone
btn.classList.contains("activo");    // true o false
✅ Buena práctica: Definid el aspecto en CSS con clases, y desde JS solo activad y desactivad esas clases. .style solo cuando el valor no podéis conocerlo de antemano.

No solo modificamos lo que ya existe. Podemos crear elementos desde cero y añadirlos al DOM. Así funcionan las listas de tareas, los chats, cualquier app.

Crear y añadir — siempre tres pasos
// 1. Crear el elemento (existe en memoria, pero no se ve)
const nuevoP = document.createElement("p");

// 2. Configurarlo
nuevoP.textContent = "Soy nuevo";

// 3. Añadirlo al DOM
document.body.appendChild(nuevoP);
Eliminar
const elem = document.querySelector("#borrar-esto");
elem.remove(); // desaparece del DOM

Un evento es cualquier cosa que ocurre en la página: un clic, mover el ratón, pulsar una tecla. JavaScript puede escuchar esos eventos y reaccionar.

// Tres partes:
// 1. El elemento a vigilar
// 2. Qué evento escuchar (entre comillas)
// 3. Qué hacer cuando ocurra (función)

elemento.addEventListener("click", function() {
  // esto se ejecuta cuando el usuario hace clic
});
Eventos más comunes
click        → el usuario hace clic
dblclick     → doble clic
mouseenter   → el ratón entra en el elemento
mouseleave   → el ratón sale del elemento
mousemove    → el ratón se mueve sobre el elemento
mousedown    → presiona el botón del ratón
mouseup      → suelta el botón del ratón
