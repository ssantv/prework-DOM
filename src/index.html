Introducción al DOM

Cómo podemos editar la página al interactuar con ella

¿Qué es el DOM?

DOM son las siglas de Document Object Model. Es la representación del HTML como un árbol de objetos que JavaScript puede leer y modificar.

El árbol del DOM:

document
  └── html
      ├── head
      │   └── title
      └── body
          ├── h1
          ├── p
          └── button
💡 La idea clave: Cuando el navegador carga tu HTML, lo convierte en este árbol de objetos. JavaScript puede recorrerlo, leer sus valores y modificarlos en tiempo real, sin recargar la página.

HTML que escribes:

<h1 id="titulo">
  Hola mundo
</h1>

<p class="intro">
  Texto aquí
</p>

Lo que JS ve:

// Un objeto con propiedades:
{
  id: "titulo",
  textContent: "Hola mundo",
  style: { ... },
  classList: [ ... ]
}

Seleccionar elementos

Antes de modificar cualquier cosa, necesitamos seleccionarla. Hay tres formas principales.

getElementsByTagName, getElementsByClassName y getElementById
// Selecciona por tag, por clases o por id
              const p = document.getElementsByTagName("p");
              const titulo = document.getElementsByClassName("mi-titulo");
const titulo = document.getElementById("mi-titulo");

// HTML necesario:
// <h1 id="mi-titulo">...</h1>
querySelector
// Selectores CSS — el más flexible
// Devuelve el PRIMERO que encuentre

document.querySelector("h1");
document.querySelector("#mi-id");
document.querySelector(".mi-clase");
document.querySelector("ul li");
querySelectorAll
// Devuelve TODOS los que coincidan
// → NodeList (se recorre con forEach)
const items = document.querySelectorAll("li");

items.forEach(function(item) {
  console.log(item.textContent);
});
⚠️ Diferencia importante:
  • getElementById("titulo") — sin #
  • querySelector("#titulo") — con #, igual que en CSS

Si el elemento no existe, ambos devuelven null. Intentar modificar null da error.

Manipulación del DOM

Una vez seleccionado el elemento, podemos cambiar lo que muestra.

textContent
const p = document.querySelector("p");

// Cambiar el texto
p.textContent = "Nuevo texto";

// Leer el texto actual
console.log(p.textContent);

// ✅ Seguro: trata todo como texto
innerHTML
const div = document.querySelector("div");

// Puede incluir HTML
div.innerHTML = "<strong>Texto importante</strong>";
¿Cuándo usar cada uno?
  • Usa textContent cuando el contenido sea texto plano.
  • Usa innerHTML solo cuando necesites insertar HTML real y controlas el contenido.
Cambiar estilos — .style
const box = document.querySelector(".caja");

box.style.color = "red";
box.style.backgroundColor = "black";
box.style.fontSize = "2rem";
box.style.display = "none"; // ocultar

// ⚠️ camelCase, no kebab-case
// background-color → backgroundColor
classList — clases CSS
const btn = document.querySelector("button");

// Añadir una clase
btn.classList.add("activo");

// Eliminar una clase
btn.classList.remove("activo");

// Alternar (si está la quita, si no la pone)
btn.classList.toggle("activo");

// Comprobar si la tiene
btn.classList.contains("activo");
✅ Mejor práctica: Usa classList en lugar de .style siempre que puedas. Define el estilo en CSS y solo activa/desactiva la clase desde JS. Así separas responsabilidades.

Crear y eliminar elementos

No solo podemos modificar lo que existe. Podemos crear elementos nuevos y añadirlos a la página, o eliminar los que ya están.

Crear y añadir
// 1. Crear el elemento
const li = document.createElement("li");

// 2. Darle contenido
li.textContent = "Nuevo item";

// 3. Añadirlo al DOM
const lista = document.querySelector("ul");
lista.appendChild(li);
Eliminar
// Seleccionar y eliminar
const elem = document.querySelector(".borrar");

elem.remove(); // ← desaparece del DOM

// También desde el padre
const padre = document.querySelector("ul");
padre.removeChild(elem);

Vigilar lo que hace el usuario

Consta de tres partes. Seleccionar el elemento y añadirle un evento, indicarle el tipo de evento que tiene que tener en cuenta y lo que debe hacer (la función).

// Eventos más comunes
click          El usuario hace clic en un elemento.
dblclick       El usuario hace doble clic en un elemento.
mousedown      El usuario presiona un botón del ratón sobre un elemento.
mouseup        El usuario suelta un botón del ratón sobre un elemento.
mousemove      El usuario mueve el ratón sobre un elemento.
mouseenter     El ratón entra en el área de un elemento.
mouseleave     El ratón sale del área de un elemento.
